Keys are sent only to this local Studio server for the current request. They are saved only if you tick Save config, which writes to your local .env.

Advanced provider settings

Leave this blank for built-in cloud providers. Only fill it for Custom OpenAI-compatible providers or if your provider gives you a different endpoint.

OpenAI-compatible providers use /chat/completions. Ollama uses /api/chat. Anthropic uses /v1/messages.

Leave unchecked to avoid writing anything. API keys are saved to provider-specific env vars such as OPENAI_API_KEY; custom providers use LLM_API_KEY.
